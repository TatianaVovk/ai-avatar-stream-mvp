# AI Avatar Stream MVP

Прототип автономного AI-стрима с несколькими агентами, ведущими научную дискуссию
без участия человека.

Проект реализован как end-to-end MVP: от генерации диалога до получения
готового видеофайла, пригодного для публикации.

Демо (YouTube, сохранённая live-сессия / VOD):  
https://youtu.be/Qj_qwuLMuMs

Один запуск пайплайна формирует готовый видеофайл.

---

## Пайплайн

1. Генерация реплик для каждой роли (LLM, сценарная логика)
2. Синтез речи с использованием разных голосов
3. Сборка видео из аудиодорожек и статичных аватаров
4. Получение финального файла для публикации

Пайплайн запускается одной командой и не требует ручных шагов между этапами.

---

## Роли агентов

В проекте реализованы три автономные роли:

- Ведущий - вводит тему и представляет участников
- Учёный - объясняет и развивает тему
- Скептик - задаёт уточняющие и критические вопросы

Контекст диалога сохраняется между репликами.

---

## Технологический стек

- Python
- OpenAI GPT-4 - генерация диалога (модель заменяема)
- OpenAI Text-to-Speech - озвучивание реплик
- MoviePy - сборка видео
- FFmpeg - кодирование и рендеринг видео
