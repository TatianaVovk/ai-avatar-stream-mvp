Прототип автономного AI-стрима с несколькими агентами, которые ведут научную дискуссию без участия человека.  
Проект реализован как end-to-end MVP: от генерации диалога до публикации видео на YouTube.

Демо (YouTube, сохранённая live-сессия / VOD):  
https://youtu.be/Qj_qwuLMuMs

---

В проекте реализован автономный диалог между двумя AI-агентами с разными ролями:
- **учёный** — объясняет и ведёт дискуссию,
- **скептик** — задаёт уточняющие и критические вопросы.

Диалог:
- генерируется моделью LLM с сохранением контекста,
- озвучивается с помощью Text-to-Speech,
- автоматически собирается в видео с использованием статичных аватаров,
- публикуется на YouTube.

Проект выполнен в формате MVP с фокусом на воспроизводимый пайплайн и инженерную простоту.

---

## Архитектура и пайплайн

1. Генерация реплик диалога между агентами (LLM, role-based логика)
2. Синтез речи для каждой реплики (Text-to-Speech)
3. Сборка видео из аудиодорожек и статичных аватаров
4. Получение финального видеофайла для публикации

Все этапы выполняются автоматически, без ручного вмешательства.

---

## Технологии и инструменты

- **Python**
- **OpenAI GPT-4** — генерация диалога с сохранением контекста
- **OpenAI Text-to-Speech** — озвучивание реплик разными голосами
- **MoviePy** — сборка видео
- **FFmpeg** — backend для кодирования видео
- **YouTube** — публикация результата (VOD)

## Структура проекта

```text

├── assets/          # обложка и общие ассеты

---
Запуск проекта:

1) Установи зависимости:
pip install -r requirements.txt

2)Создать файл .env на основе .env.example

3) Запустить генерацию диалога и сборку видео:
python src/main.py


На выходе формируется видеофайл, готовый для публикации на YouTube.
